# Portafolio Personal - Jose

Un portafolio moderno y responsive construido con React, TypeScript, TailwindCSS y Vite. Incluye internacionalización (ES/EN) y animaciones suaves.

## Tecnologías

- **Frontend**: React + Vite + TypeScript
- **UI**: TailwindCSS, Framer Motion, Lucide React
- **Router**: React Router DOM v6
- **i18n**: i18next + react-i18next
- **Linting**: ESLint + Prettier

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone <repo-url>
   cd portafolio-jose
   ```

2. Instalar dependencias:
   ```bash
   npm install
   ```

3. Ejecutar en desarrollo:
   ```bash
   npm run dev
   ```

4. Construir para producción:
   ```bash
   npm run build
   ```

## Estructura de Carpetas

```
src/
  app/          # Providers y rutas
  assets/       # Múltimedia
  components/   # Componentes UI globales (botones, cards)
  features/     # Módulos (About, Hero, Skills, etc)
  layout/       # Layout principal (Navbar, Footer)
  locales/      # Traducciones JSON
```

## Características

- Diseño Responsive (Mobile/Desktop)
- Switch de Idioma persistente
- Modo Oscuro/Claro (base preparada en CSS)
- Navegación suave
